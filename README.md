# Катя и Гаврик

Сайт для двух замечательных кошек.

## Запуск

```bash
# Production (Docker)
docker compose up -d --build
```

Открой http://localhost:80

---

## Как добавить фото в галерею

1. Положи фотографии (`.jpeg` или `.jpg`) в папку:
   ```
   Фото_котов/
   ```
2. Пересобери и запусти проект:
   ```bash
   docker compose up -d --build
   ```
3. После запуска скопируй фото в проект:
   ```bash
   cp Фото_котов/*.jpeg frontend/public/images/
   ```
4. Открой файл `frontend/src/data/galleryPhotos.ts` и убедись, что пути к новым файлам добавлены в список.

> **Важно:** Имена файлов чувствительны к регистру. Не используй пробелы и кириллицу в именах файлов.

## Как добавить видео

1. Конвертируй видео в формат **MP4 (H.264 + AAC)**. Формат H.265 не поддерживается большинством браузеров.
2. Положи файл `.mp4` в папку:
   ```
   Видео_котов/
   ```
3. Пересобери и запусти проект:
   ```bash
   docker compose up -d --build
   ```
4. После запуска скопируй видео в проект:
   ```bash
   cp Видео_котов/*.mp4 frontend/public/video-files/
   ```
5. Открой файл `frontend/src/data/videos.ts` и добавь новое видео в список:
   ```ts
   {
     id: 'video-4',
     title: 'Название видео',
     description: 'Описание видео',
     thumbnail: '/images/имя-файла-превью.jpeg',
     videoUrl: '/video-files/имя-файла.mp4',
   },
   ```

> **Важно:** Файл превью (`thumbnail`) должен лежать в `frontend/public/images/`.

## Как редактировать истории

1. Открой файл `frontend/src/data/stories.ts`
2. Каждая история выглядит так:
   ```ts
   {
     id: 'story-1',
     catId: 'katya', // 'katya', 'gavrik' или 'both'
     title: 'Заголовок истории',
     text: 'Текст истории...',
     date: '2024-01-01',
   }
   ```
3. Чтобы **добавить** историю — добавь новый объект в массив `stories`.
4. Чтобы **изменить** — отредактируй `title`, `text` или `date`.
5. Чтобы **удалить** — удали объект из массива.
6. Пересобери проект:
   ```bash
   docker compose up -d --build
   ```

> `catId` определяет, в каком разделе появится история:
> - `'katya'` — в разделе "Приключения Кати"
> - `'gavrik'` — в разделе "Приключения Гаврика"
> - `'both'` — в разделе "Как мы нашли друг друга"
