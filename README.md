# Катя и Гаврик

Сайт для двух замечательных кошек.

## Фотографии

Положи JPG фотографии в `frontend/src/images/`:
- `cat-katya.jpg` — фото Кати
- `cat-gavrik.jpg` — фото Гаврика
- `story-1.jpg`, `story-2.jpg`, `story-3.jpg` — фотографии к историям
- `gallery-1.jpg` ... `gallery-6.jpg` — галерея

## Запуск

```bash
# Development
cd frontend && npm install && npm run dev

# Production (Docker)
docker compose up -d --build
```

Открой http://localhost:80
